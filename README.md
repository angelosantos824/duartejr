# Duarte JR - Sprint de layout e conteudo

Projeto estatico da Duarte JR Transportes Especiais e Turismo.

## Como executar

1. Abrir esta pasta no VS Code.
2. Usar Live Server em `http://127.0.0.1:5502` ou outro servidor estatico local.

## Dados oficiais

- Configuracoes da empresa: `assets/js/config.js`
- Servicos: `assets/js/data/services.js`
- Frota: `assets/js/data/vehicles.js`
- Conteudo institucional: `assets/js/data/company-content.js`

## Assets principais

- Logo principal: `assets/images/logo-duarte-jr.png`
- Logo do rodape: `assets/images/logo-duarte-jr-footer.png`
- Hero principal: `assets/images/hero/hero-duarte-jr.jpg`
