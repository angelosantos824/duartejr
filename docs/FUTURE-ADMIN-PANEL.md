# FUTURE ADMIN PANEL — Proposta

Objetivo: Painel administrativo para que o proprietário atualize conteúdo sem editar arquivos.

Funcionalidades esperadas:
- Autenticação (Supabase Auth)
- CRUD de veículos (dados, imagens, disponibilidade)
- Gerir textos das páginas e banners
- Gerir contatos e número de WhatsApp
- Visualizar solicitações de orçamento
- Upload de imagens (Supabase Storage)

Planeamento técnico:
- Backend: Supabase (Auth, Database, Storage)
- Frontend: Painel responsivo com formulários e upload
- Segurança: Row Level Security configurada no Supabase

Observação: Nesta Sprint não implementar o painel. Estrutura atual evita conteúdo estático espalhado por HTML.
