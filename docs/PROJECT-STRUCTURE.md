# Project Structure

Site estatico com paginas HTML, CSS modular e dados centralizados em JavaScript.

## Paginas

- `index.html`: hero, servicos, chamada institucional, frota em destaque e CTA.
- `sobre.html`: historia, valores, missao e visao.
- `aluguel.html`: modalidades e formulario de orcamento via WhatsApp.
- `frota.html`: cards de frota filtraveis.
- `contato.html`: contatos oficiais, formulario via WhatsApp e mapa.

## Dados

- `assets/js/config.js`: dados oficiais da empresa e contatos.
- `assets/js/data/services.js`: lista oficial de servicos.
- `assets/js/data/vehicles.js`: lista de categorias e veiculos.
- `assets/js/data/company-content.js`: conteudo institucional reutilizavel.
